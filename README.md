# Campus Life Super App

A multi-page web application designed to help students navigate campus life by providing easy access to events, dining options, and a campus map.

This project demonstrates multi-page structure, interactive navigation, and clean front-end development practices.

## Features
- Multi-page navigation (Home, Events, Dining, Map)
- Interactive navigation menu
- Organized campus information display
- Responsive layout design

## Technologies
- HTML
- CSS
- JavaScript
- Git & GitHub
